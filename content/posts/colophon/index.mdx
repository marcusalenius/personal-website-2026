## Design

The design of this website is inspired by [Gabriel Sha's](https://gsha22.github.io/), [Ben Cuan's](https://bencuan.me/), [Nikhil Jha's](https://nikhiljha.com/), and [Oliver Ni's](https://oliver.ni/) websites.

It is typeset in [Nunito Sans](https://fonts.google.com/specimen/Nunito+Sans) and code is in [Inconsolata](https://fonts.google.com/specimen/Inconsolata?preview.script=Latn).

The icons are [Phosphor Icons](https://phosphoricons.com/).

## Technology Stack

It's a Next.js and React site written in TypeScript that is statically exported and served via GitHub Pages. The source code can be found [here](https://github.com/marcusalenius/personal-website-2026).
